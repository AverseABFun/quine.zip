Quine.zip

Below are two textboxes. The one on the left is for input, and the one on the right displays output.

The goal here is to write a series of commands on the left that result in the output on the right matching exactly what's on the left.

In the world of programming, such a series of commands is called a quine

The commands available to you are:

  • print M: prints the following M lines of input (they are ignored, but must be valid commands)
  • repeat M N: repeats the last M lines of output, starting N lines from the end

Only write one command per line!

Examples:

Input:
print 2 print 3 print 4 repeat 1 1
Output:
print 3 print 4 print 4
Input:
print 0 print 2 print 1 print 2 repeat 3 2
Output:
print 1 print 2 print 1 print 2 print 1
